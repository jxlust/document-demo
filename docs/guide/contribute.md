在你 clone 了 xxx 项目代码并且使用 npm install 安装完依赖后，你还可以运行下面几个常用的命令：

1. npm start 在本地运行 xx 项目

3. npm test 运行测试

4. npm run build 构建项目

1. 克隆需要贡献的项目【[项目代码仓库](https://github.com/xxx)】
